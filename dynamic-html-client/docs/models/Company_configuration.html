Company_configuration

    digital certificate A1 model, p12, encoded by base64
  • certificatepwd : String
    certificate password

    2

    how the document will be printed - '0' # without DANFE; - '4' # DANFe NFC-e; - '5' # DANFe NFC-e e-mail
    Enum:
    0
    4
    5
