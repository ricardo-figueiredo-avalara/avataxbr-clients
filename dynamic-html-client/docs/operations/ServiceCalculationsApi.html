calculationsAccountAccountIdCompanyCompanyCodeTransactionTypeDocumentCodeGet

Retrieve a single transaction

URL

http://avataxbr-sandbox.avalarabrasil.com.br/v2/calculations/account/{accountId}/company/{companyCode}/{transactionType}/{documentCode}

HTTP Method

GET

Response Type

Parameters

  • Header: Authorization String

    Bearer {auth}

  • Path: accountId UUID

    Account ID

  • Path: companyCode String

    Company Code

  • Path: transactionType String

    Transaction Type (sale, purchase, receipts or payment)

  • Path: documentCode String

    Document Code
